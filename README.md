# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/lukevinicius/focal-point.git
```

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Go to http://localhost:3000/ to see the result.
